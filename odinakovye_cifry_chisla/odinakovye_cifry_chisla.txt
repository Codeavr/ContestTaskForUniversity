Одинаковые цифры числа
На вход подается число. Определить, есть ли среди цифр введенного числа - одинаковые. Если есть, то вывести их количество с соответствующими значениями, иначе - вывести “NULL”. 

Ввод
Строка принимает число длины n.
0 < n <= 10.
Вывод
Если среди цифр введеного числа есть одинаковые - вывести их количество с соответствующими значениями, иначе - вывести “NULL”.
Формат вывода:
1)"Число1 количествоN1";
2)"Число1 количествоN1, Число2 количествоN2, Число3 количествоN3";
3)"NULL".
