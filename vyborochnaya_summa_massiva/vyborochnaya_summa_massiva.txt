Выборочная сумма массива
На вход подается массив целых чисел. Вывести сумму четных чисел. Если чётных чисел нет вывести “NULL”. 

Ввод
Строка принимает массив из n целых чисел.
0 < n <= 1000.
Вывод
Вывести сумму четных чисел. Если чётных чисел нет вывести “NULL”.
Формат вывода:
1)"SUM=Число1";
2)"NULL".
