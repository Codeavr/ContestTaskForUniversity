Удвоение переменной n раз
На вход подаются две переменные a и n. Программа умножает переменную a на 2 ровно n раз и возвращает полученный результат. 

Ввод
На вход подаются две переменные a (длины m) и n.
0 < m < 10.
0 <= n <= 10.
Вывод
Умножить переменную a на 2 ровно n раз и вывести результат.

Пример ввода и вывода:
Ввод:
5 7
Вывод:
640
